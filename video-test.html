영상 방식 테스트

같은 페이지 안에서 어떤 방식이 실제 환경에서 되는지 비교하는 테스트입니다. 모바일 기준 16:9로 맞췄고, 각각 동작 여부만 보면 됩니다.

1. 유튜브 기본 iframe

가장 일반적인 방식입니다. 환경에 따라 `153`이 날 수 있습니다.

2. 유튜브 nocookie iframe

쿠키 관련 제약을 줄이는 방식입니다. 기본 iframe보다 나은 경우가 있습니다.

3. 클릭 후 iframe 생성

처음부터 iframe을 만들지 않고, 플레이를 눌렀을 때만 삽입합니다. `153` 가능성을 줄이는 쪽입니다.

플레이를 누르면 현재 페이지 안에서 iframe이 생성됩니다.

4. 유튜브 직접 링크

같은 페이지 안 재생은 아니지만, 실제로는 가장 잘 열리는 방식입니다.

유튜브 열기

5. 같은 탭 이동

재생 버튼을 누르면 현재 탭에서 바로 유튜브로 넘어갑니다. 가장 현실적인 대체 방식입니다.

현재 탭 이동

6. 유튜브 앱 스킴 시도

유튜브 앱이 있으면 앱으로 열릴 수 있고, 없으면 실패하거나 브라우저로 처리됩니다.

유튜브 앱 열기 안드로이드 intent

7. mp4 직접 재생

가장 안정적인 현재 페이지 재생 방식입니다.
